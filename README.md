# HL Service By Hedisson Lopez

Proyecto listo para subir a Vercel o Netlify.

## Cómo usarlo en tu computadora

1. Instala Node.js 18 o superior.
2. Abre la carpeta del proyecto en la terminal.
3. Ejecuta:

```bash
npm install
npm run dev
```

## Para publicarlo en Vercel

1. Crea una cuenta en Vercel.
2. Sube esta carpeta completa.
3. Vercel detectará que es un proyecto Vite.
4. Pulsa **Deploy**.

## Archivos principales

- `src/App.jsx` → contenido de la web
- `src/styles.css` → estilos
- `index.html` → SEO básico y título

## Personalizaciones rápidas

- Cambiar textos: `src/App.jsx`
- Cambiar colores: `src/styles.css`
